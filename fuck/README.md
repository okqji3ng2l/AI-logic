# Side Tilt & Posture Detection

啟動：`python side_tilt.py`，按 `Q` 離開。首次執行自動下載模型。

---

## 偵測指標與角度限制

### 1. 軀幹傾斜（肩膀 → 髖部，相對垂直線）

| 範圍 | 狀態 |
|------|------|
| −10° ~ +10° | UPRIGHT（直立） |
| > +10° | FORWARD（前傾） |
| < −10° | BACKWARD（後傾） |

### 2. 駝背（耳朵 → 肩膀 → 髖部 夾角）

| 脊椎夾角 | 頭部前傾偏移 | 狀態 |
|----------|-------------|------|
| ≥ 155° 且偏移 ≤ 5% | — | GOOD POSTURE |
| 140°–155° 或偏移 5–12% | — | SLIGHT HUNCH |
| < 140° 或偏移 > 12% | — | HUNCHED |

> 頭部前傾偏移 = 耳朵與肩膀的水平距離 ÷ 畫面寬度

### 3. 全身傾斜（肩膀 → 踝關節，相對垂直線）
僅作數值參考，不單獨判斷狀態。
